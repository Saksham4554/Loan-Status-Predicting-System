# Loan-Status-Predicting-System
Loan Approval Prediction App

A Machine Learning-powered web application built using Streamlit that predicts whether a loan application will be approved or rejected based on applicant details.

Features
🔍 Predict loan approval instantly
📊 Uses trained Machine Learning model
🎯 Clean and interactive UI with Streamlit
⚡ Real-time prediction based on user input
📈 Scaled numerical features for better accuracy
🧠 Machine Learning Model

Model trained using historical loan dataset

Preprocessing includes:
Feature scaling (StandardScaler)
Feature selection
Model saved using Joblib

🛠️ Tech Stack
Python
Streamlit
NumPy
Pandas
Scikit-learn
Joblib
